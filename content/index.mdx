Install Dannbox using your preferred package manager:

```bash
bun add dannbox
```

## Quick Example

Here's a simple example to get you started:

```typescript
import { Dannbox } from 'dannbox'

const app = new Dannbox({
  mode: 'production'
})

await app.start()
```
